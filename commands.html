Bot Commands

Complete list of all available slash commands

Team Management

/team create
Create a new team in your league
Usage: /team create name:[team name]
/team list
List all teams in the league
Usage: /team list
/team roster
View a team's roster
Usage: /team roster team:[team name]

Player Management

/player add
Add a player to a team
Usage: /player add team:[team] player:[username]
/player remove
Remove a player from a team
Usage: /player remove team:[team] player:[username]
/player stats
View player statistics
Usage: /player stats player:[username]

Scheduling

/schedule create
Schedule a new match
Usage: /schedule create team1:[team] team2:[team] time:[datetime]
/schedule list
View upcoming matches
Usage: /schedule list
/schedule report
Report match results
Usage: /schedule report match:[id] score:[result]

Applications & Tickets

/application setup
Configure application system
Usage: /application setup
/ticket create
Create a support ticket
Usage: /ticket create subject:[subject]
/ticket close
Close a ticket
Usage: /ticket close

Verification

/verify
Verify your Roblox account
Usage: /verify username:[roblox_username]
/verify setup
Configure verification system
Usage: /verify setup

Budget & Transactions

/budget view
View team budget
Usage: /budget view team:[team]
/transaction record
Record a transaction
Usage: /transaction record type:[type] amount:[amount]

Draft System

/draft start
Start a draft
Usage: /draft start rounds:[number]
/draft pick
Make a draft pick
Usage: /draft pick player:[username]

Security

/security raid_mode
Toggle raid protection mode
Usage: /security raid_mode enable:[true/false]
/security config
Configure security settings
Usage: /security config
